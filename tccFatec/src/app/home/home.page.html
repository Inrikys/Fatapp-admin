Manual do Usuário Fatapp Admin Controlar Salas


  1. No menu, clique na opção “Salas”;
  2. Para cadastrar uma nova sala clique no ícone de “+”;
  3. Será exibida uma tela onde após o preenchimento e o clique em “Adicionar Sala”;
  4. Será redirecionado a tela anterior, com a nova sala cadastrada;
  5. Para editar a sala, basta clicar no ícone de “lápis” onde será exibido as informações para serem adicionadas à sala.

    6. A Figura ao lado representa a interface do Fatapp Admin referente ao percurso para se controlar salas.
Cadastrar Recursos


  1. No menu, clique na opção “Salas”;
  2. Para editar a sala, basta clicar no ícone de “lápis”;
  3. Clique em “Adicionar Recursos”;
  4. Preencha as informações desejadas e clique em “Adicionar Recurso”;
  5. O sistema redirecionará o usuário para a tela anterior com o recurso cadastrado.

    6. A Figura ao lado representa a interface do Fatapp Admin referente ao percurso para se cadastrar recursos.
Cadastrar Evento


  1. No menu, selecione a opção “Cadastrar evento”;
  2. Digite as informações do novo evento e clique em “Cadastrar evento”.
    3. A Figura ao lado representa a interface do Fatapp Admin e o passo a passo para se cadastrar um evento.
